This project demonstrates a monitoring setup using Prometheus, Grafana, Node Exporter, and cAdvisor.

## Architecture
- Prometheus collects metrics
- Node Exporter collects node-level metrics
- cAdvisor collects container metrics
- Grafana visualizes metrics

## Setup Steps
1. Deploy Node Exporter
2. Deploy cAdvisor
3. Deploy Prometheus
4. Deploy Grafana
5. Access Grafana dashboard

## Tools Used
- Kubernetes
- Prometheus
- Grafana
